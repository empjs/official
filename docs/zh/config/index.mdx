# 配置总览
`EMP` v3 提供了与 v2 相似的配置项，通过本章节，你可以了解 `EMP` 配置的使用方式。

import Overview from '@components/Overview';

<Overview list={OVERVIEW_CONFIGS} />

当你运行 `EMP` 的命令行工具时，`EMP` 会自动读取当前路径下的 `emp-config.js` 文件。
一个基础的 `EMP` 配置文件如下所示：

```ts title="emp-config.js"
import {defineConfig} from '@empjs/cli'
export default defineConfig(store => {
  return {}
})
```
