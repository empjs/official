# define
+ 全局环境替换
+ 类型： `Record<string, any>`

## defineFix
+ 类型 `all` | `esm` | `cjs` | `none`
+ 默认 `cjs`

:::tip 
是否创建 cjs 的 process.env 或者 esm 的 import.meta.env
all 两者创建
esm 创建 import.meta.env
cjs 创建 process.env
none 不创建
:::

```ts title="emp-config.js"
import {defineConfig} from '@empjs/cli'
export default defineConfig(store => {
  return {
        define:{
            cdnHost:'cdn.com'
        },
        defineFix:'cjs'
    }
  }
})
```
